# Siniestralidad_ZMM_Merida
Visor web elaborado con código html que tiene como objetivo mostrar un estudio sobre la sinistralidad en Municipios de la Zona Metropolitana de Mérida, Yucatán. El presente trabajo muestra el desarrollo de un proyecto utilizando información del Marco geoestadístico de INEGI 2025, la Red Nacional de Caminos 2025 y los registos georreferenciados de la ATUS para el año 2024 con la infromación sobre los sinistros viales. Con base en la información obtenida se elaboraron capas con índices y categorías de riesgo vial, a partir de modelos de mapas de calor usando la severidad de los accidentes como campo ponderador para generar los clusters, conteo de accidentes por medio de herramientas de localización espacial vectorial y creación de campos con sentencias SQL para agilizar la elaboración de índices como campos categóricos. El resultado son capas vectoriales con las clasificaciones de accidentes por colonias en las que se elaboraron categorías en función del tipo de accidentes, así mismo para la capa de accidentes se categorizaron en función de la severidad y para las vialidades se hizo lo mismo considerando una distancia de 15 metros del accidente y ponderando según la severidad de estos mismo.
